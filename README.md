# SL-project-management

AI-augmented project management dashboard with interactive Gantt charts, risk scoring and automated status reports.

## Features

- Interactive Gantt chart with task dependencies and milestone tracking.
- Multi-project dashboard with KPI overview and progress metrics.
- Resource allocation analysis across projects and team members.
- AI-powered status report generation via Groq LLM.
- CSV import for project data with sample templates included.
- Bilingual interface (FR/EN).

## Installation

```bash
git clone https://github.com/Erikeo29/SL-project-management.git
cd SL-project-management
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```
