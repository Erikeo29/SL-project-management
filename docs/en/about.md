# About

## Augmented project management

Streamlit application for project tracking and steering with artificial intelligence.

### Objective

Provide a lightweight and deployable tool to:
- Visualize the progress of a project portfolio
- Automatically detect delay risks
- Optimize resource allocation
- Generate AI-powered status summaries

### Technical stack

| Component | Technology |
|-----------|------------|
| Interface | Streamlit |
| Visualization | Plotly |
| Data | Pandas |
| AI | Groq API (LLaMA 3.3 70B) |
| Deployment | Streamlit Cloud |
