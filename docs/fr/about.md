# A propos

## Gestion de projet augmentee

Application Streamlit de suivi et pilotage de projets avec intelligence artificielle.

### Objectif

Fournir un outil leger et deployable pour :
- Visualiser l'etat d'avancement d'un portefeuille de projets
- Detecter automatiquement les risques de retard
- Optimiser l'allocation des ressources
- Generer des syntheses de statut par IA

### Stack technique

| Composant | Technologie |
|-----------|-------------|
| Interface | Streamlit |
| Visualisation | Plotly |
| Donnees | Pandas |
| IA | Groq API (LLaMA 3.3 70B) |
| Deploiement | Streamlit Cloud |
